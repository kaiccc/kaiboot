## kai4boot

爬虫信息推送到微信，关注最新的商品、影视信息。主要是懒得打开网站一点点找。。。

商品：备战双11  获取到最新的商品活动（爬的 smzdm数据）

影视：爬的 mp4ba

后续还会增加一些网站的爬虫。暂时没想好。
